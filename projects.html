PROJECTS

PDX Transit

This app won the first place award at the 2014 FOSS4G international conference in the best web map application category. FOSS4G is the largest global gathering focused on open source geospatial software. Using TriMet Web Services and Open Street Map, it displays TriMet routes and stops and arrival time.

Recycle City Chicago

This app won the first prize of the Apps for Metro Chicago, Delta Green Opportunities App Award Competition! You can check the Apps for Metro Chicago website for more information and all other apps submitted for this event. This app shows the closest recycling center and the driving direction to that location to help residents of Chicago recycle.

World Census Map

This displays World Census information of a country. You can check Age distribution, Life Expectancy, Female/Male ratio, Birth/Death/Net Migration per 1,000 persons, Infant and under 5 year Mortality Rate per 1,000 persons. All data is from U. S. Census Bureau APIs - International Data Base. This API provides Time Series International Data Base by 5-Year Age Groups and Sex.

San Diego Kids Education Info

The SD Kids Edu Info application helps residents in the San Diego County find nearby Schools and Daycare centers, and obtain information about schools ranking, dropout rate, graduation rate, average class size, number of students and School Districts information.

Iowa City Bus Tracker

This app displays bus routes and stops of Iowa City, Coralville, and University of Iowa bus systems so people can check all bus route/stop data in one map application. It finds the closest bus stop from a user's location and shows predictions for upcoming bus arrival by using xml data feed from real-time passenger information system - Bongo

Residential Build-Out Analysis

- Meeting infrastructure demand using GIS

A fast-growing county used GIS for predicting the number and location of housing units and residents so it could anticipate demands on infrastructure. The process was captured in ModelBuilder and can be reused with new data or for different geographic areas.

Forsyth County eStatus

The map portion of this site provided locations and basic information of land development permits and public hearing applications. The location information of all applied permits and applications are updated into the database for this web site every day from the production GIS database using python script.

Iowa Natural Features

This map displays natural features in Iowa State. It uses Google Map API v3 and the data is downloaded from USGS U.S. Board on Geographic Names site in 2011. Reservoir, Forest, Island, Lake, Mine, and Swamp were chosen as major natural features and displayed in the map. Each feature can be toggled on and off and the list of features are provided as a table next to the map. If you click the feature on the table, an information box will show up and the map will zoom into that feature.

Gresham Neighborhood Finder

This app displays City of Gresham Neighborhood District Boundary and provides a tool to find out which neighborhood district you are in by address
